
# Python Crawl4AI
> Includes support for websites, docs and tweets.

# Setup
```bash
docker build -t <name> ./
docker run -p 8000:8000 <name>
```

# Test
> Visit to [Docs](http://localhost:8000/docs)
> Click on `Try it out`
> Enter `q` as `https://www.google.com`
> Enter `depth` as 1 
> Hit `Execute`
