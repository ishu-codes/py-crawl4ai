
# Python Crawl4AI
> Includes support for websites, docs and tweets.

# Setup
```bash
docker build -t <name> ./
docker run -p 8000:8000 <name>
```

# Test
> 1. Visit to the [Docs](http://localhost:8000/docs)
> 2. Click on `Try it out`
> 3. Enter `q` as `https://www.google.com`
> 4. Enter `depth` as 1 
> 5. Hit `Execute`
